表面粗さとは

表面粗さは、加工した面の微細な凹凸の大きさを数値で表したものです。図面では面の機能（しゅう動・シール・はめあい・外観など）に応じて上限値を指示し、加工側はそれを満たす加工法を選びます。日本では JIS B 0601（製品の幾何特性仕様＝GPS、表面性状）で定義され、国際規格 ISO 4287 / 21920 に対応しています。

Ra と Rz の違い

Ra（算術平均粗さ）は、基準長さ内で粗さ曲線の平均線からの高さの絶対値を平均した値です。面全体の平均的なザラつきを表し、最も広く使われます。一方 Rz（最大高さ粗さ）は、基準長さ内で最も高い山と最も深い谷の合計で、局所的な突起や深い傷を捉えるのに向きます。

注意：新旧 JIS で Rz の定義が違う。 現行 JIS B 0601（2001 年版以降）の Rz は「最大高さ粗さ」（旧規格の Ry に相当）です。 これに対し、1994 年版 JIS の Rz は「十点平均粗さ」（現在の Rzjis）で、意味が異なります。 古い図面の「Rz」を読むときは、引用している規格年を必ず確認してください。比較や換算の前提が変わります。

カットオフ λc と評価長さ

粗さ測定では、うねり成分を除くために カットオフ値 λc を設定します。λc は測りたい Ra の値域に応じて JIS B 0633 / ISO 4288 で標準値が決まっており、評価長さは原則 λc の 5 倍です。本ツールは入力した Ra から標準 λc を自動表示します（この値は規格表に基づく確定値です）。

粗さの選び方

面の機能から必要な粗さを決め、過剰に滑らかにしないのがコツです。滑らかすぎる指示は加工コストを跳ね上げます。

  • はめあい・しゅう動面：研削レベル（Ra 0.4〜1.6）が目安。精密なすべり軸受はさらに細かく。
  • シール面（O リング・ガスケット）：漏れ防止のため Ra 0.8〜3.2 程度が一般的（シール種で指定確認）。
  • 一般的な機械加工面：Ra 3.2〜6.3 で十分なことが多い。
  • 非接触・外観のみ：Ra 12.5〜25 など粗めでコストを抑える。

はめあい公差とセットで考えると判断しやすいです（関連ツール参照）。

使い方

  • 標準数列ボタンか直接入力で Ra を指定します。
  • 右側に 標準 λc・評価長さ（規格値）、Rz の目安、旧 ▽ 記号の目安、その Ra を狙える加工法が表示されます。
  • Rz 換算の係数は手で調整できます（既定 4）。

よくある質問

Ra と Rz はどう違う？
Ra は面の平均的なザラつき、Rz は最も高い山と深い谷の合計で局所的な突起を捉えます。目的で使い分けます。
Ra から Rz は計算できる？
厳密な換算式は規格にありません。Rz ≒ 4×Ra が目安ですが 3〜7 倍の幅があり、確定値が要るなら実測です。
カットオフ λc はどう決める？
JIS B 0633 で Ra の値域ごとに標準値が決まっています（例：Ra 0.1〜2µm → λc 0.8mm）。本ツールが自動表示します。
旧図面の Rz は今と同じ意味？
違うことがあります。1994 年版の Rz は十点平均粗さ（現 Rzjis）です。規格年を確認してください。
▽記号と Ra の対応は？
旧仕上げ記号の目安で、▽▽▽ で Ra 0.8〜1.6 程度。現行図面では Ra 値で指示します。

対応範囲

Ra 標準数列 0.012〜100 µm。標準カットオフ λc は JIS B 0633 の区分（Ra 0.006〜80 µm）で表示します。Rz の換算値、旧 ▽ 記号、加工法の対応は目安であり、規格で一意に定まる値ではありません。
