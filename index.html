はめあい公差 計算

基準寸法とはめあい記号（H7/g6 など）から、穴・軸の許容寸法、すきま／しめしろ、はめあい種別を公差帯ダイアグラム付きで計算。穴基準・軸基準の両対応（JIS B 0401 / ISO 286）。

ツールを開く 公開中 TOOL 02

ボルト締結 計算

呼び径・強度区分・潤滑条件から、締付トルクと軸力を相互換算。引張応力と降伏に対する利用率をゲージで表示（メートル並目・ISO 898）。

ツールを開く 公開中 TOOL 03

ねじ下穴径（タップ下穴）

メートルねじの下穴径を、呼び径・ピッチ・ひっかかり率から計算。標準推奨（呼び径−ピッチ）と早見表付き。並目・細目に対応。

ツールを開く 公開中 TOOL 04

表面粗さ 換算・早見

Ra（算術平均粗さ）から標準カットオフ λc・評価長さ、Rz の目安、旧仕上げ記号 ▽、対応する加工法をまとめて表示。新旧 JIS で異なる Rz の定義も解説（JIS B 0601 / B 0633）。
